# RemoteControl_RTOS
基于FreeRTOS和STM32F103RCT6的无线遥控手柄

##硬件部分
硬件环境：嘉立创EDA
硬件版本：Spherical_Robot_RC_V2.0

##软件部分
软件环境：Keil v5
软件版本：RemoteControl_Released
